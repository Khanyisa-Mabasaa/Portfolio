# Khanyisa Mabasa Portfolio

A responsive AI software developer portfolio for Khanyisa Mabasa, built with HTML, CSS, and JavaScript.

## Live Sections

- Hero with robotic corporate avatar
- Recruiter-ready skills section
- Featured projects and case studies
- Experience timeline
- Built-in portfolio assistant
- Certificates
- Contact links and downloadable CV

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Static assets

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload everything in this folder.
3. Go to `Settings` > `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and `/root`.
6. Save and wait for GitHub Pages to publish the site.
